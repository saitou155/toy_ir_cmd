toyircmd

Implementation of USB Infrared Toy V2 command mimicking btoircmd

Getting Started

This is software for Ubuntu.

Prerequisites

install libusb for build.

sudo apt-get install libusb-1.0-0 libusb-1.0-0-dev

How to use
